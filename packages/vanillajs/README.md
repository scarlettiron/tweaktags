# @tweaktags/vanillajs

The full edit in place UI for plain HTML and JavaScript, no framework.

Part of [TweakTags](https://github.com/scarlettiron/tweaktags), a lightweight edit in place content layer for React, Next, and
plain HTML sites. Mark content with a `data-tweaktags-*` attribute and signed in editors change
it right on the page.

## Install

```sh
npm install @tweaktags/vanillajs
```

## Usage

```html
<script src="https://unpkg.com/@tweaktags/vanillajs@1.0.0"></script>
<script>
  TweakTags.init({ apiBasePath: '/api/tweaktags' });
</script>
```
